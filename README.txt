
TRANSLATOR:

- Translator application translates the given text into various language.

- It uses google module googletrans for modulation.

- It also able to convert the translated language to audio format.

- We can also able to hear the voice using play button.


FEATURES OF TRANSLATOR:

- Conversion of text from one language to another.

- Text to Speech conversion.

- Dark Mode.

- Remove garbage audio files frequently.
